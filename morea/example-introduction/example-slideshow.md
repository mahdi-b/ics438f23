# Slideshow Example

This page demonstrates how you can embed a slideshow. In this example, we embed a SlideShare

The following markdown:

{% raw %}
https://speakerdeck.com/oembed.json?url=https://speakerdeck.com/jnunemaker/atom
{% endraw %}

produces this gist:

https://speakerdeck.com/oembed.json?url=https://speakerdeck.com/jnunemaker/atom
