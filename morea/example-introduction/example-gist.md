# GitHub Gist Example

This page demonstrates the use of [GitHub Gists](https://www.mathjax.org) for including code snippets in markdown files.

The following markdown:

{% raw %}
{% gist philipmjohnson/5288814 %}
{% endraw %}

produces this gist:

{% gist philipmjohnson/5288814 %}
